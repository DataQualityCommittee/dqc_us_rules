Q1.In the case below where the breakdown of available securities has subtotals should the company use the Investment Type Axis to represent those subtotals of Amortized cost?

In this case the values for short term investments and cash equivalents should be tagged using the FinancialInstrumentAxis with a member to represent the subtotal. The member representing the subtotal should be nested with children of the members representing the financial instruments comprising that sub total. This ensures that the values reported for the amortized cost will all use the axis FinancialInstrumentAxis and the concept AvailableForSaleDebtSecuritiesAmortizedCostBasis.

Q2. In the example above the company has reported cash as part of the investment schedule. Should the value of 849 be tagged with the CashMember on the FinancialInstrumentAxis or should it be tagged with the concept Cash?

The value of cash should not be not be included in the aggregate total of AvailableForSaleSecuritiesDebtSecurities or AvailableForSaleDebtSecuritiesAmortizedCostBasis. In the case above the value of $5,992 should be tagged with CashCashEquivalentsAndShortTermInvestments. The value of $849 should be tagged with the concept Cash. For the amortized cost including cash an extension element will have to be used. Many companies have defined extension elements for this, however the standard extension element CashAndAvailableForSaleDebtSecuritiesAmortizedCostBasis should be used.

Q3. Should securities representing Cash and cash equivalents be included as part of AvailableForSaleDebtSecuritiesAmortizedCostBasis?

Yes.
